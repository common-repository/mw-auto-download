== Description ==

### Create automatic download links ###

Auto download is the easiest way to create automatic download links. Start a download from any page.


### About Auto Download

It's a simple plugin for Wordpress ideal for creating download pages. The following options are currently available:

* Base auto download links on own classname
* Exclude pages from using the plugin
* Create links with the auto download option
* Set time before auto download start for individual pages
* Create multiple auto downloads on a single page


### How it works

1. Set up the settings for your auto download links.
2. Add classname to link or check the box 'Auto download' when creating a link.

or

1. Place the shortcode [mw-download] into the Wordpress editor.
2. Place the url text between the shortcode. (ex. [mw-download]Text here[/mw-download].
3. Add the "url" parameter with the url to the content or page.
